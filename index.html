Sublime Text 2 右键菜单设置

sublime text 2 是个很性感的编辑器 呵呵~
如果是安装版 通常会自动添加 右键菜单 Open with Sublime Text 2
但是如果是用的绿色版 就没有这个福利了
沒有福利咱们自己动手丰衣足食
1.添加右键菜单
win+r 启动运行 输入 Regedit
找到 'HKEYCLASSESROOT\*\shell'
添加新建项“Open With Sublime Text 2”,再在Open With Sublime Text 2 下面新建项command
在command 下面新建__字符串值
数据填写

D:\tools\Sublime Text 2\sublime_text.exe “%1”

保存.完成添加右键菜单

2.给Open with Sublime Text 2添加Icon
添加完右键菜单 总感觉那里怪怪的 原来是菜单上缺少Icon
找到Open With Sublime Text 2 的注册表所在项
新建 字符串值 名称为Icon 类型REG_SZ 数据填写

D:\tools\Sublime Text 2\sublime_text.exe,0

好了这样就添加上Icon了.

PS:注意是在Open with Sublime Text 2 下建立新的 字符串值 不是在command下
